# Secure WireGuard VPN — Security Engineering Case

##  Описание
Проект демонстрирует проектирование и защиту VPN-инфраструктуры
на базе WireGuard с учётом реальных сетевых и NAT-проблем.

Проект выполнен в учебных и исследовательских целях.

---

##  Используемые технологии
- WireGuard
- Linux (Ubuntu/Debian)
- Docker
- iptables / nftables
- NAT traversal
- MTU tuning

---

##  Архитектура
VPN-сервер развернут в Docker-контейнере.  
Клиенты подключаются из сетей с NAT и мобильных провайдеров.


---

##  Безопасность
- Используется только аутентификация по ключам
- Ограничены `AllowedIPs`
- Отключён root-доступ по паролю
- Реализован `PersistentKeepalive`
- Исключено хранение секретов в репозитории

---

##  Разбор инцидента
В процессе эксплуатации возникла проблема:
- соединение устанавливается
- трафик не проходит / нестабильный ping

**Причина:**  
MTU mismatch + особенности NAT

**Решение:**
- откат глобального MTU
- корректная настройка `wg syncconf`
- `PersistentKeepalive` на клиентах

Подробности: `incident/connectivity-issue.md`
